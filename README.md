Requirements

    Node.js 20.x

Installation
bash
Copy

git clone https://github.com/stark0908/Xero.git
cd Xero

# Install dependencies
npm install
pip install -r requirements.txt

# Initialize blockchain components
aptos init --network devnet

# Start services
docker-compose up -d

# Launch platform
cd dapp && npm run dev
